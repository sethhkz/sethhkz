## 📂 Featured Projects

### 🏠 [Home Lab Network — Pi-hole & DNS Filtering](https://github.com/sethhkz/home-lab-networking)
Built a Raspberry Pi-based home lab with Pi-hole DNS filtering, monitoring 14,000+ live DNS queries with 18.6% blocked. Used PingPlotter and router diagnostics to investigate latency, packet loss, and channel-level signal issues (T3 timeouts, QAM downgrades).

### 💻 [VMware Virtualisation Lab](https://github.com/sethhkz/vmware-virtualization-lab)
Multi-OS virtualisation lab running Windows and Kali Linux VMs simultaneously. Hands-on configuration of virtual hardware, NAT vs bridged networking, and isolated cross-platform testing environments.

### 🔒 [Secure Remote Access Infrastructure](https://github.com/sethhkz/secure-remote-access)
Full secure remote access setup using Tailscale VPN, SSH, RDP, SMB file sharing, and Wake-on-LAN — across 8 devices on Windows, iOS, and Linux. Built without using unsafe port forwarding.
